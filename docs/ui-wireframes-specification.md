### 组件详细设计

#### 顶部状态栏
```
高度：56px
背景：深色渐变 (#1A1A1A → #2D2D2D)

左侧区域 (40%)：
- 用户头像：32px 圆形
- 用户名：14px，白色
- 等级标识：Lv.XX，12px，金色

右侧区域 (60%)：
- 金币显示：图标 + 数值
- 钻石显示：图标 + 数值
- 体力显示：图标 + 当前/最大值
- 间距：每项之间 12px
```

#### 用户信息卡片
```
高度：80px
背景：卡片背景，圆角 12px
内边距：16px

左侧：
- 用户头像：48px 圆形，品质边框
- 在线状态指示器：绿色圆点

中间：
- 用户名：16px，字重 600
- 等级进度条：经验值显示
- 公会信息：公会名称（如有）

右侧：
- VIP等级：VIP图标 + 等级
- 战力值：总战力显示
```

#### 活动横幅
```
高度：120px
圆角：12px
轮播间隔：5秒
指示器：底部圆点

内容：
- 背景图：活动主题图片
- 标题：18px，字重 700，白色
- 描述：14px，白色，半透明
- 按钮："立即参与"，主要按钮样式
```

#### 功能入口网格
```
网格：2列 × 3行
间距：12px
卡片尺寸：等宽，高度 80px

每个功能卡片：
- 背景：卡片背景，圆角 12px
- 图标：48px，居中
- 标题：16px，字重 600，居中
- 副标题：12px，灰色，描述信息
- 红点提示：新功能或未读消息
```

#### 召唤动画设计
```
动画流程：
1. 按钮点击 → 资源扣除确认
2. 召唤门打开动画 (1秒)
3. 光芒特效 (0.5秒)
4. 武将卡片飞入 (0.8秒)
5. 品质揭示动画 (1秒)
6. 结果展示 (持续显示)

特效等级：
- 普通/优秀：简单光芒
- 稀有：蓝色光芒 + 音效
- 史诗：紫色光芒 + 震动
- 传说：橙色光芒 + 特殊音效
- 神话：红色光芒 + 全屏特效
```

## 🎯 交互设计规范

### 手势操作
```
点击：选择、确认
长按：显示上下文菜单
滑动：翻页、滚动
拖拽：移动武将、排序
双击：快速操作
捏合：缩放地图
```

### 反馈机制
```
视觉反馈：
- 按钮按下：缩放效果
- 加载状态：骨架屏
- 成功操作：绿色提示
- 错误操作：红色提示

触觉反馈：
- 重要操作：轻微震动
- 错误操作：强震动
- 成功操作：短震动
```
